Developed a machine learning pipeline to classify phage lytic proteins for therapeutic applications, converting unstructured biological data into vectorized inputs. Engineered high-dimensional embeddings from ~2,500+ enzyme sequences using the PhaLP 2.0 database and the 565 MB ESM-2 model.
